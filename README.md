# Python Scripts Collection

This repository contains a collection of Python scripts for interactive web apps, data visualization, web scraping, and algorithmic problem-solving.

## Execution Guide

Follow the steps below to set up the environment and run the respective scripts.

### 1. Install all required dependencies & Run

```bash
pip install streamlit numpy matplotlib beautifulsoup4 requests
streamlit run number-guessing-game.py
# Plot Dow Jones Industrial Average (DJIA) data
python Plotting-DJIA.py

# Plot student performance results
python Plotting-students-results.py

# Run web scraping and admission evaluation logic
python scraping&admission-logic.py

# Calculate the depth of a binary tree
python depth-of-binary-tree.py

# Compare two binary trees
